ng-app :- 
Auto bootstrap angular application'
Designates root of the element.


If an application if there is only one ng-app directive available then we need not to trigger it manually it will automatically execute.
however, if there are multiple ng-app directives available in a HTML page then we need to trigger it manually (requires manual bootstrap).


ng-init :-
create initialization code.
Evaluate expression.
Do not use it.( notta a best practice)


ng-if :-
remove or re-creates a portion of DOM based on an expression.
similar to if-else condition.

ng-bind :-
bind a particular variable to the tag where this attribute define.

{{}} :-
another alternative to use variable and expression.

Register the controller in that module
